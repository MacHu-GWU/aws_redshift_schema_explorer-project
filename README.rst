Install
------------------------------------------------------------------------------

``aws_redshift_schema_explorer`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install aws-redshift-schema-explorer

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws-redshift-schema-explorer
